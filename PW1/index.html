JavaScript

O que é JavaScript e qual é sua principal função no desenvolvimento web?

O JavaScript é uma linguagem de programação de alto nível usada para criar, controlar e alterar o comportamento dinâmico de elementos em uma página web. Ele permite a implementação de recursos complexos e interativos, transformando sites estáticos em aplicações dinamicas.

Qual a diferença entre let, const e var para declarar variáveis?

const: Possui escopo de bloco e não permite reatribuição (o valor não pode ser alterado após ser declarado). É o padrão recomendado para quase todas as variáveis. let: Possui escopo de bloco, mas permite reatribuição. Usado quando você sabe que o valor da variável precisará mudar no futuro. var: É a forma mais antiga e problemática. Possui escopo de função/global e permite reatribuições e redeclarações. Hoje em dia, o uso de var não é recomendado

Qual é a diferença entre == e ===?

== (Igualdade Ampla): Avalia se dois valores são iguais. Se forem de tipos diferentes, a linguagem tenta converter um deles para o mesmo tipo antes de comparar (ex: 1 == "1" true). === (Igualdade Estrita): Avalia se os valores são idênticos e se pertencem ao mesmo tipo de dado. Se os tipos forem diferentes, o resultado é sempre falso (ex: 1 === "1" é false).

O que são closures e para que elas servem?

Uma closure (ou fechamento) é a capacidade de uma função lembrar e acessar variáveis do seu escopo externo, mesmo que essa função externa já tenha terminado de ser executada. Em essência, a função "embrulha" a si mesma e ao seu ambiente léxico.

O que é o event loop e como ele funciona no JavaScript?

O Event Loop (ou Loop de Eventos) é o mecanismo central que permite ao JavaScript realizar operações assíncronas e não bloqueantes, mesmo sendo uma linguagem de thread única (single-thread), o que significa que ela executa uma única instrução por vez.

Python

O que é Python e quais são suas principais vantagens?

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Criada para priorizar a legibilidade e a simplicidade, ela permite que desenvolvedores escrevam sistemas complexos utilizando menos linhas de código do que em linguagens tradicionais.

Como funciona o gerenciamento de memória em Python?

O gerenciamento de memória em Python é automático. Ele cuoca da alocação de espaço para objetos e da sua liberação quando não são mais necessários, utilizando principalmente dois mecanismos: Contagem de Referências e Coleta de Lixo

O que são classes e objetos em Python?

Em Python, classes são moldes ou plantas estruturais usados para criar objetos. Uma classe define as características (atributos) e os comportamentos (métodos) que os objetos terão. Um objeto é uma instância específica desse molde, contendo dados reais e podendo executar as ações definidas.

O que são herança e polimorfismo?

1. Herança O que é: Cria uma hierarquia onde uma nova classe é baseada em uma classe já existente. A classe filha ganha todos os atributos e métodos da classe pai, evitando repetição de código. Exemplo prático: Imagine que você tem uma classe base chamada Animal com a ação emitirSom(). A partir dela, você pode criar as classes filhas Cachorro e Gato. Ambas herdam a estrutura básica de um Animal, mas têm suas próprias características adicionais. 2.polimorfismo O que é: Do grego "muitas formas", é a capacidade de chamar o mesmo método em objetos diferentes e obter comportamentos específicos para cada um deles. Exemplo prático: Seguindo o exemplo acima, se você mandar os objetos Cachorro e Gato executarem o método emitirSom(), o cachorro vai "latir" e o gato vai "miar". O programa sabe exatamente qual comportamento executar dependendo do objeto que foi chamado, utilizando o mesmo nome de método.
